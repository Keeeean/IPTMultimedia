How does a computer work?
Let's go through it!
CPU stands for the Central Processing Unit

This is the piece of hardware in the computer that enables it to interact with applications and programmes

Simply put, it is the component of the computer that opens applications and executables. Most games are executables!

RAM, or otherwise known as memory stands for the Random Access Memory

This is the piece of hardware in the computer that allows open application a place to store and access data on a short-term basis.

For example, you're probably using a web-browser to look at this website. The web-browser takes up storage in your RAM to access the data that is on this site!
